PROJECT SHOWCASE

Brain Tumor Detection Using Hybrid ResNet-50 and VGG16 with Attention Mechanism

This final year project focuses on developing an intelligent brain tumor detection system using deep learning techniques. The system combines ResNet-50 and VGG16 convolutional neural networks with an attention mechanism to improve classification performance on MRI brain images.

The main purpose of this project is to assist medical practitioners in identifying and classifying brain tumors more efficiently. The project involves image preprocessing, feature extraction, model training, and performance evaluation. It demonstrates the application of artificial intelligence, computer vision, and machine learning in the healthcare field.

MediMeal Assist

MediMeal Assist is a web-based healthcare and nutrition management system designed to help users manage healthy eating habits and monitor nutritional intake. The system provides features such as user registration, meal planning, nutrition tracking, and nutritionist management.

This project demonstrates full-stack web development skills using PHP, MySQL, HTML, CSS, JavaScript and CodeIgniter 4. It also focuses on user-friendly interface design, database management, and practical healthcare-related software development.
